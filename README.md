# Baby Blocks: A Level Editor for *Baby Steps*

**Baby Blocks** is an unofficial [MelonLoader](https://melonwiki.xyz/) mod that adds a full in-game **level editor** to *Baby Steps*. Fly around the world, place and edit props pulled straight from the game's own asset catalog, build materials and custom accessories, and save your creations as custom levels.

It's highly recommended that you use the Thunderstore Mod Manager to install this mod!

## Features

- **Works with the [Baby Steps: Multiplayer Mod](https://github.com/caleborchard/Baby-Steps-Multiplayer-Mod-Client)** — edits, prop placement, materials, grouping, physics modes, and custom accessories all sync live between players for collaborative, real-time level building
- **Prop placement** — browse and place any prop from the game's catalog, organized in a searchable, categorized palette
- **Transform gizmos** — move, rotate, and scale props easily with on-screen handles, plus precision numeric entry in the properties panel
- **Grouping** — combine props into groups and transform them as a unit, with group root position, rotation, and display scale all preserved
- **Custom accessories** — turn any prop into a wearable hat or held/grabable item, with hair, offset, sunglasses, and passthrough tuning
- **Terrain Holes** — carve holes into the base game's terrain by dragging in the Terrain Hole prop
- **Spawn points** — place where players start in your level
- **Physics modes** — toggle per-prop physics behavior, including freezing in place until disturbed
- **Base Map toggle** — hide the base game world to build on a clean slate
- **Undo / redo** — full editor history
- **Save / load custom levels** — save your builds, browse them in a custom levels menu, and load them back in
